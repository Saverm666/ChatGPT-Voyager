# ChatGPT-Voyager

一个可直接本地加载的 Chrome 扩展，为 ChatGPT 和 Notion 提供一些高频效率增强功能。

## 有什么功能

### ChatGPT

- 分支提问：在 ChatGPT 里选中一段回答后，可以直接基于这段内容发起新的分支提问，不用手动复制、粘贴、加引用
- 段落复制为 Markdown：复制回答内容时，会尽量保留标题、列表、引用、代码块和公式等 Markdown 结构，拿去记笔记或继续加工更顺手
- 长对话优化：为超长 ChatGPT 会话提供右侧时间线、快速跳转和重点标记，减少反复滚动查找上下文
- 公式点击复制：公式直接点一下就能复制。支持 LaTeX、UnicodeMath (Word)和纯 LaTeX 多种公式格式。
- 公式历史记录：最近复制过的公式会保存在本地，方便再次复用
- 提示词收藏：保存和管理常用提示词，在 Popup、设置页和页内入口中一键复制
- 输入增强：支持改成 `Enter` 换行、`Ctrl+Enter` 发送，更适合长 prompt 和多段文本

### Notion

- Notion 页面保护：关闭、刷新或跳转前给出确认提示，减少误操作
- Notion 公式转换：把 `$...$` 和 `$$...$$` 转成 Notion 原生公式，并在粘贴后尝试自动处理

## 怎么安装

1. 下载或克隆本仓库到本地
2. 打开 Chrome，进入 `chrome://extensions/`
3. 打开“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择当前仓库根目录

代码更新后，在扩展管理页点击“重新加载”，再刷新目标页面即可。
